Каталог coins_model_1 - сохранённая обученная модель, не актуальна.
Каталог animals - содержит датасеты с картинками животных
Каталог images - содержит проверочные картинки с животными
Каталог output - содержит обученную модель по распознаванию картинок
Каталог CoinsVGG - содержит исходный код модели, распознающей монетку, содержит следующие файлы:
 - image_renamer.py - скрипт, котороый переименовывает картинки из датасета по нужному модели формату
 - predict.py - скрипт, который обращается к ранее обученной модели, и передаёт изображение для проведения анализа и распознавания
 - smallvggnet.py - обучаемая модель нейронной сети
 - train_vgg_coins.py - скрипт, который обучает модел
